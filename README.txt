# Test from Gridnine company.

## Steps
1) Set directory in GitBash and clone into it project from GitHub repo.
git clone https://github.com/lenz86/gridnine_task.git

2) Start IntellijIdea, open project from directory.

3) Import JUnit dependency from maven if you don't use auto-import.

4) Mark "com.gridnine.testing" Directory as Source root

5) Mark "com.gridnine.test.java" Directory as Test source root

###
You can find questionary.odt file with answers on questions into resources directory.
